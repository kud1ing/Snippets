# Rust WebAssembly client using Trunk with Hot Reloading

## Instructions

* make sure that [Rust is installed](https://www.rust-lang.org/tools/install)
* make sure that [Trunk is installed](https://trunkrs.dev)
* run `trunk serve --open`
